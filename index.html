~$ cat sample_rule.pseudo

Illustrative pseudocode for a time-based blind SQLi detection. Not proprietary client content.

~/detections/sqli_time_blind.pseudo 
01  # RULE: Time-Based Blind SQLi — aggregation + anti-evasion
02  # MITRE ATT&CK: T1190 — Exploit Public-Facing Application
03
04  rule "sqli_time_blind_v2" {
05    source   = waf_logs | apache_access
06    trigger  = request.query matches
07               /(SLEEP|BENCHMARK|WAITFOR\s+DELAY|pg_sleep)\s*\(/i
08    where    = response.time_ms > 1500
09    aggregate = 3 events / 60s / source_ip
10    suppress  = known_scanners | health_checks
11    severity = HIGH
12    notify   = soar.playbook("sqli_response_v3")
13  }
14
15  # False-positive guard: many WAFs fingerprint themselves in UA.
16  # Chain with contextual asset criticality before paging L3.
